# I18n Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the console and website choose Chinese or English from the user's system language, allow manual switching, persist the choice, and align the UI with the generated mockups.

**Architecture:** Add a small browser-side i18n module with `localStorage` persistence and browser-language fallback. The console imports it for all visible labels; the static website uses the same storage key and redirects between existing localized pages.

**Tech Stack:** Vanilla ES modules, embedded Go static assets, Playwright browser E2E, existing CSS.
